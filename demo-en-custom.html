Example of customized English "+2 widget"

Here we customize the default English settings like this: 
    $('#share-me').socialSharePrivacy(window.ssp_settings_en(
      {
        'media_prefix':'http://thedod.github.com/socialshareprivacy/socialshareprivacy/',
        'services': {
          'facebook': {
            'action':'like', 'dummy_img':'images/dummy_facebook_en_like.png',
            'perma_option': 'on'
          },
          'twitter': {'perma_option': 'on'},
          'gplus': {'perma_option': 'on'}
        },
      }));
  • media_prefix is now defined as an absolute url. Originally, it assumed media was under socialshareprivacy/ relative to the current page\ (and it kinda sucked). Note that yyou don't need the http://.... A media_prefix can simply start with a /
  • Facebook has a "like" button instead of "recommend" (we also need to change dummy_img for this).
  • User can auto-enable widget for all 3 services (this is stored as site-wise cookie).
    Note: In the original German version this is enabled by default, but I took the liberty to disable it by default in the English version, because I don't think it's a good idea :)
