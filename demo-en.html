Example of English "+2 widget" with default setting

The English version of the German Social Share Privacy widget (aka "+2 widget") contains the German distribution as is (to make it easier to merge upstream changes).

The only addition is a the file ssp_settings_en.js providing the function ssp_settings_en() that generates English-language settings for the widget. In this example we use it without parameters: 
    $('#share-me').socialSharePrivacy(window.ssp_settings_en());
